Just some general useful information for working your way around these folders. 

content -> talk contains the files for the recent and upcoming talks. 

The content folder is where most of the files for the design of the website are contained

The textbook is found at static -> files -> _book

The introduction information is found at myWebsite -> content -> authors -> admin
